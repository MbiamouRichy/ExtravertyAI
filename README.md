## 🚀 À propos de Extraverty AI

Ce dépôt contient le code source du site vitrine d'**Extraverty AI**, une agence en ligne dédiée à la transformation du service client via des solutions d'automatisation WhatsApp avancées. 

Le site a pour objectif de présenter nos services, de détailler le fonctionnement de nos workflows (basés notamment sur des architectures n8n optimisées) et d'exposer nos offres tarifaires :
- **Starter** (15 000 FCFA)

- **Business** (55 000 FCFA)

🌐 **Site en direct :** [https://www.extravertyai.com](https://www.extravertyai.com)

## 🛠️ Stack Technique

Le projet a été développé avec une stack moderne axée sur la performance, le typage strict et une expérience utilisateur (UX/UI) haut de gamme.

- **Framework :** [Next.js](https://nextjs.org/) (App Router pour un rendu hybride et optimisé)
- **Langage :** [TypeScript](https://www.typescriptlang.org/) (Pour la robustesse et la maintenabilité)
- **Styles :** [Tailwind CSS](https://tailwindcss.com/) (Pour un design sur-mesure et responsive)
- **Composants UI :** [shadcn/ui](https://ui.shadcn.com/) (Utilisation du thème "neutral" pour une esthétique minimaliste et professionnelle)

## 🎨 Design System & UI/UX

L'interface a été rigoureusement pensée pour refléter un positionnement premium et technologique :
- **Palette et Contraste :** Le design intègre des environnements sombres avec un arrière-plan spécifique défini à `rgb(25, 25, 25)` sur les sections clés, sur lequel notre logo est mis en valeur en blanc pur (format SVG privilégié) sans aucune altération de ses proportions.
- **Esthétique :** Approche épurée garantissant que l'attention de l'utilisateur reste focalisée sur la valeur apportée par l'automatisation.

## 📦 Installation et Lancement en local

Pour exécuter ce projet localement sur votre machine :

1. **Cloner le dépôt :**
   ```bash
   git clone [https://github.com/MbiamouRichy/ExtravertyAI.git](https://github.com/MbiamouRichy/ExtravertyAI.git)
   cd ExtravertyAI
   
2. Installer les dépendances :

Bash
npm install
# ou
yarn install
# ou
pnpm install

3. Lancer le serveur de développement :

Bash
npm run dev
# ou
yarn dev
# ou
pnpm dev

4. Ouvrez http://localhost:3000 dans votre navigateur pour visualiser le site.
